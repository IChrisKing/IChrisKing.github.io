配置方法：

  1. 选择New profile
  2. 起个名字，比如ss
  3. Protocol选择SOCKS5
  4. Server 填写127.0.0.1
  5. Port填写 1080
  6. 點擊左側的Apply changes
  7. 接下來訪問網頁的時候，點擊地址欄右邊的圓圈，選擇剛纔新建好的ss，就可以了
